# Day 1 Scope Freeze

Date: 2026-03-30

This note freezes the Day 1 objective for the edge-AI roadmap.

## Scope

- runtime lock
- VisDrone bootstrap
- baseline training launch
- investor positioning brief
- handoff/status capture for Day 2

## Explicit non-goals

- GCS/dashboard cleanup
- supplementary dataset ingestion
- synthetic data generation
- ONNX or TensorRT export
- hardware prototype work
- investor list building
- outreach or deck drafting

## Runtime policy

- Primary runtime: RTX 4060 machine via WSL2 Ubuntu
- Fallback runtime: Mac local `.venv` with Python 3.11 and MPS
- Shared verification path: `scripts/verify_training_runtime.py`

## Day 1 success bar

Day 1 is successful if:

1. at least one runtime is verified as training-capable
2. `data/visdrone_police` exists and is audited
3. a baseline `yolo26n` training run is launched
4. one investor-facing positioning brief is written
5. the Day 2 start point is written down with no ambiguity
